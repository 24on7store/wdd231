Individual Project Site Plan

Site Name

MaUniverse – This name represents a unified digital brand offering services like MaService (web development), Echo Talk (podcast), and 4U Store (e-commerce). It's modern, broad, and emphasizes versatility in one universe.

Site Purpose

The purpose of this website is to serve as a hub for various creative and professional services including motivational podcasts, web development, and an online marketplace—all under the MaUniverse brand.

Scenarios

  • How can I book web development services through MaService?
  • Where can I listen to the latest episodes of Echo Talk?

Color Schema

  • Primary Color (Indigo - #3F51B5): Used for headings, navigation bar, and buttons.
  • Accent Color (Amber - #FFC107): Used for call-to-action buttons, icons, and highlights.
  • Text Color (Dark Gray - #212121): Main body text.
  • Secondary Text (Gray - #666666): Descriptions or metadata.
  • Background (White - #FFFFFF): Page background.

Typography

  • Poppins: Used for all headings (h1, h2, h3).
  • Open Sans: Used for all body text.
